# delivercalc
Обычный калькулятор
Доступен ввод через кнопки калькулятора или клавиши клавиатуры

Посчитать сумму:
  кнопка "="
  клавиша "Enter"
Очистить полностью
  кнопка "СЕ"
  клавиша "ESC"
Очистить поле
  кнопка "С"
  клавиша "Backspace"

Сложение / Вычитаение / Деление / Умножение / Степень / Подсчет со скобками
  Соответсвующие кнопки 
  Соответсвующие клавиши
